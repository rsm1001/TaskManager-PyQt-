# TaskManager-PyQt

基于 PyQt6 和 SQLite 的现代化任务管理系统，改进自原有的 Tkinter 版本。

## 功能特点

- **现代化界面**: 使用 PyQt6 构建，界面更美观
- **SQLite 数据库**: 使用 SQLAlchemy ORM 管理数据，性能更好
- **任务分类**:
  - 每日必做任务（支持按星期分类）
  - 待办事项（带截止日期和紧急程度计算）
  - 娱乐任务（多种娱乐类别）
- **数据导入导出**: 支持 JSON 格式导入导出
- **智能提醒**: 待办事项按紧急程度加权随机抽取
- **每日重置**: 自动重置每日任务完成状态
- **新增功能**: JSON导入示例显示 - 在帮助菜单中可查看各类任务的JSON格式示例

## 项目结构

## 安装和运行

### 方法一：使用虚拟环境（推荐）

```bash
python launch.py
```

脚本会自动创建虚拟环境并安装依赖。

### 方法二：手动安装

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
python main.py
```

## 功能说明

### 每日必做任务
- 可按星期分类（每天、星期一至星期日）
- **新增**: 界面提供星期筛选功能，默认显示今天是星期几
- 自动每日重置未完成的任务状态
- 支持添加、编辑、删除、完成标记

### 待办事项
- 支持设置截止日期
- 自动计算紧急程度（越临近截止日期权重越高）
- 已过期任务会有特殊标识

### 娱乐任务
- 多种娱乐类别（游戏、电影、运动等）
- 支持娱乐计划管理

### 数据管理
- SQLite 数据库存储，更加稳定可靠
- JSON 格式导入导出，兼容旧版数据
- 数据备份和恢复功能

### 新增功能：JSON导入示例显示
- 通过顶部菜单栏的"帮助"->"JSON导入示例"可查看各类任务的JSON格式示例
- 包含每日任务、待办事项和娱乐任务的详细JSON结构说明
- 便于用户准备正确的JSON文件格式进行数据导入

## 主要改进

相对于原有 Tkinter 版本，本次重构包含以下重要改进：

1. **技术栈升级**：
   - 从 Tkinter 迁移到 PyQt6，提供更现代化的界面
   - 从 JSON 文件存储升级到 SQLite 数据库，使用 SQLAlchemy ORM
   - 更好的数据持久性和查询性能

2. **数据管理改进**：
   - 引入了专业的数据库管理机制
   - 保留了JSON导入导出功能，确保与原系统的兼容性
   - 包含了数据迁移脚本，方便从旧系统升级

3. **功能增强**：
   - **新增**: 每日任务的星期筛选功能（这是针对用户反馈的改进）
   - **改进**: 星期筛选默认显示今天是星期几（这是针对用户第二次反馈的改进）
   - **新增**: JSON导入示例显示功能（便于用户理解JSON格式）
   - 更清晰的界面布局和用户体验
   - 更丰富的表格视图，支持多列排序和筛选
   - 保持了原有的核心功能（任务管理、随机抽取、紧急度计算等）

4. **架构改进**：
   - 模块化设计，职责分离更明确
   - 配置管理集中化
   - 代码可维护性和扩展性更强

## 键盘快捷键

- `Ctrl+N`: 添加每日任务
- `Ctrl+T`: 添加待办事项  
- `Ctrl+E`: 添加娱乐任务
- `Delete`: 删除选中任务
- `F5`: 刷新数据

## 技术栈

- **GUI**: PyQt6
- **数据库**: SQLite with SQLAlchemy ORM
- **数据交换**: JSON
- **随机算法**: 加权随机选择

## 开发说明

项目完全使用面向对象设计，分为几个主要模块：

1. **models.py**: 定义数据库模型
2. **data_manager.py**: 处理数据操作和业务逻辑
3. **main.py**: GUI 界面和交互逻辑
4. **json_examples_dialog.py**: JSON示例显示对话框

这种架构使得代码易于维护和扩展。
